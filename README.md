# DWM Build 1.0
My First build of DWM! 

## Dependencies
 - Xlib headers (find all these).
 - Alsamixer
 - xwallpaper
 - picom / another composite manager
 - pywal
 
 # Features / Keybinds
 - Resize tiles w/h
 - Cool Autostart.
 - Pywal to change colour scheme dynamically.
 - Vanitygaps
 - Alpha patch + pywal.
 - Use slstatus to show system status on top bar.
 - terminal scrolling
 - (check) multimedia keys support.
